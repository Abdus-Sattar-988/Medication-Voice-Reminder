# Medication Voice Reminder

## Description
Medication Voice Reminder is a professional healthcare-focused project designed to help elderly individuals take their medications on time. The system uses automated voice calls to remind users of their scheduled medication. If the user does not respond, an instant notification is sent to a caregiver (e.g., family member) to ensure proper follow-up.

## How it Works
- Elderly users receive automated voice calls at scheduled times.
- The call says: "Hello [User Name], this is your medication reminder. It is time to take your [Medication Name]. Please press 1 to confirm."
- If the user does not respond, the system sends a notification to the caregiver.
- Twilio Voice API is used to send calls reliably.

## Key Features
- Ensures timely medication adherence for elderly individuals.
- Sends instant notifications to caregivers if the user misses a reminder.
- Uses Twilio Voice API for automated, reliable calls.
- Simple and easy-to-use interface.

## Sample Voice Call
"Hello John, this is your Medication Voice Reminder. It is time to take your blood pressure medicine. Please press 1 to confirm. If you do not respond, a notification will be sent to your caregiver."
